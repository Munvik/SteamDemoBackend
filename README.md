# SteamDemoBackend

I made this project for education of several technolgies and backend basics.

It is a simple Steam-like backend where I practice:

## Why this project exists

This project was created to improve backend skills in:

- building REST APIs with ASP.NET Core
- using `FluentValidation` in application flow
- integrating `Redis` for distributed caching
- containerizzing the app with `Docker`
- preparing the codebase for `CI/CD` workflows

## Tech stack

- `.NET 8`
- `ASP.NET Core Web API`
- `MediatR` (CQRS-style handlers)
- `FluentValidation`
- `Entity Framework Core` + `PostgreSQL`
- `Redis` (`IDistributedCache`)
- `Docker`
- `CI/CD` basics

Main idea is to improve my backend skills step by step, not to build a production app.
